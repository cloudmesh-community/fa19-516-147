3.0 Abstract Database Management API

3.1 Overview:

Abstract Database Management projects provides ability to perform database operations as a service using Open API connexion service and reading specification from yaml file designed based on existing NIST template.

database.yaml for the database abstraction project is designed keeping relational SQL databases as main focus. Relational database terminology is being used.

Functions to performed database operations are defined in a python file.Three main components of database service are :

  • server.py

  • database.yaml

  • database.py

database.yaml for the database abstraction project is designed keeping relational SQL databases as primary focus. Relational database terminology is being used in api design.

3.2 SQL and NoSQL at a glance:

Key points SQL database :

  • Data records are stored in a table in form of rows and columns.

  • A set of tables makes up a schema

  • A number of schemas create a database

  • Many databases can be created on a single server

SQL and NoSQL terminologies:

NoSQL: MongoDB, CouchDB, Hbase, Cassandra DB , Oracle No SQL database etc.
